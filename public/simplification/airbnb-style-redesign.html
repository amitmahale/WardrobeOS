Today Closet Style Plan

Cleaner mobile first, cleaner desktop second.

The mobile version should feel like a practical daily companion: one recommendation, one reason, one next action. The desktop version can still expose analysis, but it should be progressively disclosed.

Airbnb-like

White surfaces, confident whitespace, soft object cards, pill controls, restrained copy, and one obvious action per panel.

Professional

Less neon, fewer heavy glass panels, fewer dense tables on primary screens, and more hierarchy between daily tasks and advanced diagnostics.

Functional

The recommender, GPT workflow, packing planner, buy-next engine, and reliability tools remain intact, but they are grouped by user intent.
